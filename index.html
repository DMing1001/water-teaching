🌊 HEC-RAS 软件概览

HEC-RAS（Hydrologic Engineering Center's River Analysis System）由美国陆军工程兵团（USACE）水文工程中心开发，是一款专业的河流水力学分析系统。

  • 一维恒定流水面线计算 — 标准步推法（Standard Step Method）
  • 一维非恒定流模拟 — 完整圣维南方程组求解
  • 二维非恒定流模拟 — 浅水方程（SWE）/ 扩散波方程
  • 泥沙输移与河床演变 — 一维和二维移动河床计算
  • 雨水管网模拟 — 管道网络（Pipe Networks）建模
  • 水温与水质模拟

HEC-RAS 自 6.0 版本起仅支持 64 位 Windows（7/8/10/11）。新版本可与旧版本并行安装，完全兼容旧项目文件。

📘 用户手册（一维 · User's Manual）

一维用户手册是 HEC-RAS 最核心的操作文档，涵盖从项目创建到结果输出的完整工作流。

主要章节

  • Getting Started — 软件安装、界面介绍、项目管理
  • Geometric Data — 河道断面编辑、河段连接、桥梁/涵洞/堰/闸门几何定义
  • Steady Flow Data — 恒定流边界条件、流量输入、水面线计算
  • Unsteady Flow Data — 非恒定流边界条件、初始条件、时间步长设定
  • Running a Model — 计算执行、收敛检查、诊断信息
  • Viewing Results — 水面线图、断面属性表、过程线、动画回放
  • Hydraulic Design — 桥梁/涵洞水力设计工具

📗 二维用户手册（2D User's Manual）

二维用户手册详细讲解 HEC-RAS 二维水动力模拟的完整工作流程，是进行洪水漫滩、溃坝、城市内涝等二维分析的必备文档。

  • 2D Flow Area Concepts — 二维计算区域概念、网格类型
  • Mesh Generation — 网格生成与细化，Breaklines 和 Breakpoints
  • 2D Boundary Conditions — 流量/水位/内边界条件
  • Running a Model — 并行计算、Numerical Reproducibility
  • 2D Results — 水深/流速/水位可视化、Reference Line 输出
  • Combined 1D/2D Modeling — 一维与二维耦合
  • Dam Breach Analysis — 溃坝分析

🗺️ RAS Mapper 用户手册

RAS Mapper 是 HEC-RAS 内置的 GIS 系统，用于管理地理空间数据、创建二维网格、查看结果和生成地图输出。

  • Terrain Data — 地形数据导入（DEM、TIN）、地形修改
  • 2D Mesh Editing — 在地图上直接编辑二维网格
  • Mapping Results — 结果的空间可视化渲染
  • Reference Lines — 参考线输出（平均流速、水力深度等）
  • Map Output — 地图导出与打印

📋 版本更新说明

HEC-RAS 6.7 Beta 5（2025.10.31）

主要修复 Beta 4 Bug，新增功能：

  • 🔧 管道网络增强 — Side Inlets、Top Inlets（替代 Drop Inlets）、Surcharge Only（模拟有盖检修孔）、管道闸门（Sluice Gate，支持 Rules/Time Series/Elevation 控制）、涵洞进水口改进
  • 🐧 Linux 支持 (WSL) — 通过 WSL 运行 Linux 计算引擎，用于云端蒙特卡洛模拟
  • 📊 数值可重现性 — 三级选项：默认（最快）、严格、最严格
  • ⚡ 多方案运行 — 自动测试不同处理器数量的运行效率
  • 🌊 堰方程能量坡度选项 — SA/2D Connections 可选 Energy Grade
  • 🌉 2D 桥梁冲刷导出 (Beta) — 导出至 FHWA Hydraulic Toolbox
  • 📈 绘图改进 — 水位/流量过程线绘图器大幅改版
  • 🌀 二次流效应 — 2D 泥沙弯道二次流，经 Koch & Folkstra (1980) 验证
  • 🔄 地理参考涵洞动量 — SWE 中包含涵洞出流动量（需手动开启）

HEC-RAS 6.7 Beta 4

  • CPU 亲和性 — 区分 P-core / E-core，计算仅用 P-core
  • 二次流 (Beta) — 弯道螺旋流模拟（平衡/非平衡方法）
  • 结构物剖面图 — 剖面中显示水工建筑物
  • 时间序列绘图 — 过程线集成表格快速切换
  • USGS 下载 — 支持按数据集过滤

HEC-RAS 6.7 Beta 2/3

  • 2D 桥梁 — 压力流与越顶流新方法
  • 1D 桥梁动量法 — General 方法（推荐）替代 Legacy
  • RAS Mapper — 菜单重组，结果访问更直观
  • 参考线变量 — 新增平均/最大流速、水力深度

HEC-RAS 6.7 Beta 1

  • 管道新管型 — 拱形、椭圆形、ConSpan 等
  • 管道剖面 — 流量/流速/临界深度剖面图
  • Speed Draw — 自动在端点放置节点
  • 形态加速因子自动化
  • BSTEM 更新 — 修复多层算法 Bug
  • 变浓度泥石流指南
